Installation
------------
1. Install Paybox cgi.

2. Copy the folder named 'paybox' and its contents to the contrib modules
   directory of your Drupal installation (for example 'sites/all/modules/').
   
3. Go to 'admin/build/modules' and enable Paybox.

4. Go to 'admin/settings/paybox' and edit options for the Paybox module.

5. Use this API in your module (see API.txt)

6. Test before use on production !
