Installation
------------
1. Copy the folder named 'paybox' and its contents to the contrib modules
   directory of your Drupal installation (for example 'modules/').

2. Go to 'admin/modules' and enable Paybox.

3. Go to 'admin/config/Paybox' and edit options for the Paybox module.

4. Use this API in your module (see API.txt)

5. Test before use on production !
